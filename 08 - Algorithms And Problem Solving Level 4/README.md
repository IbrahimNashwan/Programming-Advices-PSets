## 📂 Practical Projects (Functional Systems)
At this stage, the focus transitioned from standalone scripts to building **Integrated Functional Systems** that share data and core logic.

### 🏦 [Project 1: Bank System Extension - Version 2](https://github.com/ammmrrr18-cloud/08-Algorithms-Level-4-Bank-Extension-2)
A radical development of the core Bank System, representing a major leap in system engineering.
* **Core Logic:** Implementation of complex **User Permission Logic** (Bitmasking) to control user access to various system features.
* **Management:** Added advanced administrative features to manage system users alongside client data records.

### 🏧 [Project 2: ATM System Simulation](https://github.com/ammmrrr18-cloud/08-Algorithms-Level-4-Project-2-ATM-System)
A realistic simulation of an ATM interface that interacts directly with the Bank's central data.
* **Core Logic:** Focused on secure user authentication, real-time balance management, and transaction operations (Withdrawal/Deposit).
* **System Integration:** Demonstrates how multiple interfaces (Admin Bank System vs. User ATM System) can operate harmoniously on the same data source.
